Controle de estoque

Software pensado para gerenciar o estoque de loja, a ideia principal é que o usuário consiga gerenciar e consultar seu estoque em diferentes franquias, adicionar novos itens e fazer pesquisas de produtos em banco de dados.
Programa totalmente desenvolvido em Python utilizando a biblioteca Pyside6 e com seu banco de dados por meio do Sqlite3.

Controle de funcionários

Software com o propósito de auxiliar no controle de funcionários, podendo ser acessado como gerente para cadastro de funcionário, consultas de horários e funcionários, exclusão e afins; Ou acessado como funcionário para registro de entrada e saída.
Software desenvolvido totalmente em Python.

Simon Game

Simon Game, conhecido no Brasil também como Genius, onde a cada rodada uma nova cor aleatória entre as 4 presentes é ativada, o jogador então precisa memorizar e selecionar todas as cores que foram lançadas até o momento.
A ideia não é nada em especifico, apenas apareceu como um desafio em uma aula e eu pessoalmente gostei muito de fazer, então decidi compartilhar aqui.
o código foi escrito utilizando HTML, CSS e JavaScript e está no ar neste momento, clique Aqui para jogar.
